# jellycat

 This is my store!

 It's a React + Vite app 

 You can view the live preview here:
https://jellycat-store.netlify.app/

Or run it locally! 
1. Clone the repo
2. ```sh
   npm install
   ```
3. ```sh
   npm run dev
   ```
   to open in your localhost
4. ```sh
   npm run build
   ```
   to open thorugh the dist directory
